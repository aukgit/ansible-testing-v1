# ansible-testing

ansible-testing

## Clone

`git clone https://github.com/aukgit/ansible-testing-v1`

## Run Code

### Ubuntu / Linux / Mac

- `sh ./run.sh`

### Windows

- powershell `./run.bat`
- cmd `run.bat`

### Run using makefile

`make run-{chapter}-{os}`
For example, chapter 00, windows:
`make run-00-win` -> Will run `00. Install Ansible\makefile` with defaultTask
